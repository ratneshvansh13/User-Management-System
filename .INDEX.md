## ğŸ” Quick Answers

**Q: How do I start the app?**
A: `python app.py` then open `http://localhost:5000`

**Q: What are the login credentials?**
A: Username: `admin`, Password: `admin123`

**Q: Can I change the admin password?**
A: Yes, edit the default insert in [app.py](app.py)

**Q: How many users per page?**
A: 10 (configurable in [app.py](app.py))

**Q: Where is the data stored?**
A: In [users.db](users.db) (SQLite)

**Q: Is it responsive?**
A: Yes, works on mobile, tablet, and desktop

**Q: Can I deploy it?**
A: Yes, see [SETUP.md](SETUP.md#deployment)

**Q: What if I forget the password?**
A: Delete [users.db](users.db) and restart (recreates default admin)
