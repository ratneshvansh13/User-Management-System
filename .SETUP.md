## Troubleshooting

**Q: Port 5000 is already in use**
A: Change port in `app.py`: `app.run(debug=True, port=5001)`

**Q: Database is locked**
A: Delete `users.db` and restart the application

**Q: Flask command not found**
A: Install Flask: `pip install flask`

**Q: Invalid email error when adding user**
A: Check email format (example@domain.com)

**Q: Forgot admin password**
A: Delete `users.db` to reset everything

To deploy this application:

2. Add `runtime.txt`: `python-3.9.0`
3. `heroku create your-app-name`
4. `git push heroku main`

1. Upload files to PythonAnywhere
2. Create a web app
3. Configure WSGI file
4. Enable your web app

1. Run `python app.py` on your machine
